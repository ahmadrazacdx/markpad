*Overleaf for people who think in Markdown.*

MarkPad is a local-first desktop writing app with project structure, fast editing, and live PDF preview.

## Why MarkPad

- Markdown-first workflow with modern editing UX
- Real-time PDF preview while you type
- Project/file organization with version history
- Desktop-native installation and offline-ready operation

## Install

1. Open [Releases](https://github.com/ahmadrazacdx/markpad/releases).
2. Download the latest Windows installer (`.exe`).
3. Run the installer and launch MarkPad.

## Stack

- Tauri 2 desktop shell
- React + Vite + CodeMirror frontend
- Express + WebSocket backend
- pandoc + typst rendering pipeline
- SQLite (libSQL) + Drizzle ORM
